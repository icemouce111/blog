# Hello World

这是我的第一篇博客文章。这个个人博客使用以下技术栈搭建：

## 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite 6
- **样式**: Tailwind CSS 4 + shadcn/ui
- **路由**: React Router v7
- **内容**: Markdown 文件管理
- **评论**: Giscus (GitHub Discussions)
- **部署**: Cloudflare Pages
