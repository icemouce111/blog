# 给 OpenCode 的提示词 — 博客悬浮目录功能实现

复制下面整段话，粘贴到 OpenCode 的输入框，按回车。

***

```
在我的博客项目 blog.icemouce.cc 中实现 Markdown 文章页的悬浮目录功能。项目技术栈：React 19 + Vite 8 + Tailwind 4 + react-markdown + remark-gfm + react-router-dom + zustand。

具体要做：

1. 安装依赖：npm install rehype-slug

2. 新建 hooks/useActiveHeading.ts，实现一个自定义 hook：
   - 接收 headings 数组（{ id, text, level }[]）
   - 用 IntersectionObserver 监听标题元素进入视口
   - rootMargin 设为 '-80px 0px -60% 0px'（顶部留导航栏空间，底部 60% 不触发）
   - 返回当前激活的标题 id

3. 修改 BlogPostPage.tsx：
   - import rehypeSlug from 'rehype-slug'
   - import { useState, useEffect } from 'react'（如果还没引入）
   - 在 ReactMarkdown 组件上加 rehypePlugins={[rehypeSlug]}
   - 在 sidebar 渲染部分，用 useActiveHeading 获取 activeId，给当前激活的标题项加高亮样式（text-foreground font-medium），其余用 text-muted-foreground hover:text-foreground
   - 保持已有的 paddingLeft 层级缩进逻辑（(heading.level - 1) * 12px）

4. 修改 AiDailyPostPage.tsx：同 BlogPostPage 的改动

5. 改完后 npm run build 验证，确保零错误。

注意：
- 不要删除已有代码，只在现有基础上加功能
- 不需要悬浮浮窗（那是 OpenCode 编辑器专属功能），只需要 sidebar 滚动高亮
```
